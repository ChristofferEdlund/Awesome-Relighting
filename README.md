## Datasets

[3DV2024] Objects With Lighting: A Real-World Dataset for Evaluating Reconstruction and Rendering for Object Relighting https://arxiv.org/pdf/2401.09126

### VIDIT dataset
训练：300个游戏引擎场景每个场景40种光照(5种色温、8种方向)   
测试&验证：45个场景  
### Multi-Illumination dataset
训练：985个室内场景每个场景25种光照  
测试&验证：30个场景  
### DPR dataset
人脸数据集，利用CelebA-HQ的30000张人脸5种光照合成138,135张图像  
提供数据集制作脚本
